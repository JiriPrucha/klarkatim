A little pastel video-gift page <3
====================================

HOW TO USE
----------

1) Drop your video into THIS folder and name it:  video.mp4

   (If your file is called something else, like "birthday.mp4",
    open index.html in Notepad and look for the bright "EDIT"
    banner near the bottom -- change "video.mp4" to your filename.)

2) Want to change the message above the button?
   Open index.html in Notepad. Near the top you'll see a banner
   that says: "EDIT THE MESSAGE BELOW".
   Change the text between the > and < and save.

3) To open the page yourself: just double-click index.html.

4) To send it to him:
   - Select the whole "sister-gift" folder
   - Right-click -> "Send to" -> "Compressed (zipped) folder"
   - Send him the .zip file
   - He unzips, double-clicks index.html, presses the button. <3

NOTES
-----

- Works best in modern browsers (Chrome, Edge, Firefox, Safari).
- The page needs internet the first time it opens (to load the
  fonts). After that it caches them. If you want it to work
  fully offline, let me know and I can bundle the fonts in.
- Press Escape (or the X button) to close the video.
